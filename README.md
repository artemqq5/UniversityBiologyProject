# UniversityBiologyProject
<b>This is an abstract presentation of a cell model</b>
- The program implements an abstract model of a cell and its behavior during division
- A program using the principles of OOP and work with threads in the implementation of kotlin coroutines, also handling formatting errors during user interaction

The main purpose of the program is to show the basic behavior of a cell during division.

<h3>Приклад роботи програми</h3>

Код програми складається з таких компонентів:

Абстрактний клас Cell, який містить основні складові клітини, такі як ядро, цитоплазма, клітинний центр, ендоплазматична сітка, апарат Гольджі, лізосоми та мітохондрії. Він також містить абстрактний метод cellDivision, який відповідає за ділення клітини.

Клас CellImpl, який успадковує від абстрактного класу Cell. Він реалізує метод cellDivision, який дозволяє клітині ділитися задану кількість разів, повертаючи масив нових клітин.

Функція main, яка використовує клас CellImpl для створення прикладу клітини, отримує від користувача число разів, скільки необхідно провести ділення клітини, та виконує ділення за допомогою методу cellDivision.

Спочатку ми створюємо екземпляр клітини за допомогою класу CellImpl, передавши у конструктор основні складові клітини. Зверніть увагу на конструктор CellImpl, який приймає екземпляр класу Cell та передає його складові основному конструктору.

Далі, у функції main, ми зчитуємо від користувача число разів, скільки потрібно провести ділення клітини. Це число обмежено значенням від 1 до 5 для оптимального використання пам'яті. Після отримання введеного користувачем числа, ми викликаємо метод cellDivision клітини, передаючи число разів ділення та масив, що містить початкову клітину.

Я створюю клітину та викликаю метод ділення у неї, вказуючи кількість (2).
<img src="https://user-images.githubusercontent.com/52855607/231112411-b1cf1110-1006-4d0a-b732-c437e7ecdaee.png" width="1000">

Метод cellDivision рекурсивно розділяє кожну клітину у масиві задану кількість разів. Кожен раз, коли клітина ділиться, вона створює нову клітину з тими ж основними складовими, що й вихідна клітина. Зауважте, що нові клітини мають різні ідентифікатори, але однакові ідентифікатори для їхніх складових. Це демонструє поведінку реальних клітин під час ділення, оскільки кожна похідна клітина є окремим новим організмом, який успадковує ДНК батьківської клітини.

Тут ми бачимо візуально як поділилася клітина, вона взяла батьківські компоненти клітини з тим самим індикатором, що говорить про перейняття генетичної складової, проте, при цьому змінився основний індикатор, бо клітина новоутворена.
<img src="https://user-images.githubusercontent.com/52855607/231113137-d7a48082-3a62-444a-baa5-99d88c384e8e.png" width="1000">

Після завершення ділення ми виводимо результати, показуючи кількість створених клітин та їхні властивості. У цьому прикладі ми можемо бачити, як абстракція та принципи ООП використовуються для моделювання клітини та її поведінки ділення мовою програмування Kotlin.

Загалом, ця програма демонструє, як можна використовувати принципи абстракції та ООП для моделювання біологічних процесів, таких як клітинне ділення. Використання абстрактних класів та успадкування дозволяє нам створити гнучкі моделі, які можуть бути адаптовані для різних типів клітин або інших біологічних сутностей.
